# Food Delivery Admin App

A food delivery admin application designed to manage the entire system including products, users, orders, and loyalty programs. This app provides full control to administrators to monitor and manage operations efficiently.

---

## Features

- Admin authentication and secure login  
- Add, update, and delete food products  
- Manage product categories  
- View and manage customer orders  
- Update order status (Pending, Processing, Delivered, Cancelled)  
- Manage user accounts  
- Monitor invoices and transactions  
- Handle loyalty program tiers:
  - Bronze
  - Silver
  - Gold
  - Platinum  
- Apply and manage discounts for users based on membership level  

---

## Core Functionality

- Centralized dashboard for system control  
- Real-time order management  
- Product inventory management  
- User management system  
- Invoice and transaction monitoring  
- Loyalty and discount system management  

---

## Technology Stack

- Mobile Framework:Flutter
- Backend: FireStore
- Database: Supabase

---

## Project Purpose

This project is built to simulate a real-world food delivery administration system where admins can efficiently manage users, products, and orders while maintaining full control over the platform.
